Facial Recognition with Simplicity

My mental development from handwritten digits to your face

The whole neural network thing

When I was in school, I was first introduced to Visual Recognition by the Katana MNIST project on Kaggle. The project asks competitor to detect handwritten digit in Kannada, a language that is widely spoken in southwestern India. With convolutional neural network. There were 60 000 observation in the training set and the test set has 5000. The CNN I created is only complicated enough to detect 98.25% accuracy. This got me wonder is how hard would it be to identify faces?

Maybe I have skipped some details. The number I uses only has 784 pixels, while the normal phone camera right now has more than 8mp, which means, there are many things a machine can learn from face that consist of million pixels. Also, the Deep Learning technology that I used is very primitive.

A great lesson that I learn during the tech transition is that, if you want to learn something, learn little from a surface, and skip to the top. That totally sabotaged the ideology that I built while studying rigorous math proof in graduate school but it makes sense. For that reason, I swallow my pride and I looked up the rich library resources on python to find out that OpenCv is what people use these days for Visual Recognition. However, I discovered that OpenCV does not like Mac, it glitches badly after display a image and took me one whole day to fix the bug..

A following day, I attended TAMUHack, a Hackathon that gathers many big tech companies in the US. I visited one of their workshops about Google Cloud Visual API and that was a bomb. Everything I knew about visual recognition was outdated. Google technology is so sophisticated that it is impossible for the best programmer in the world to replicate. The number one reason is GPU and data resources. There are _ NN trained parallelly, each use a Tesla V100, a $6000 GPU that outperformed most of the data mining GPU nowadays, So, I spent another week to figure out that these crazy technologies came with a cost. It not only charges the user money per API call, it is also limited on tailoring different use for personal projects. I put a hold on the cloud service and brainstorm on what I actually want to do, and here is what I came up with.

A pair of glasses that can remember my friends name and remind me the next time I meet them.
